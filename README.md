## 🫧 About OceanBlue

**OceanBlue** is a modern web application for a sparkling blue fresh drink brand — crafted to bring the refreshing feel of the ocean straight to your screen. From product browsing to ordering, OceanBlue delivers a smooth, bubbly digital experience that matches every sip.

Whether you're discovering new flavors, building your cart, or tracking your order, OceanBlue makes it effortless and visually immersive.

---

## ✨ Features

- 🛒 **Product Catalog** — Browse all OceanBlue sparkling drink variants with detailed flavor profiles
- 🎨 **Immersive UI** — Ocean-themed animations and visuals that feel as fresh as the drink itself
- 📦 **Order Management** — Easy cart and checkout flow
- 🔍 **Flavor Explorer** — Filter and search drinks by taste, category, or ingredient
- 📱 **Responsive Design** — Fully optimized for mobile, tablet, and desktop
- 🌙 **Dark/Light Mode** — Switch between themes for any vibe

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) `>= 18.x`
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

```bash
# Clone the repository
git clone https://github.com/RaoMicoryn/oceanblue.git

# Navigate to the project directory
cd oceanblue

# Install dependencies
npm install
```

### Running the App

```bash
# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

The app will be running at `http://localhost:5173` by default.

## 🎨 Design Philosophy

OceanBlue's interface is inspired by the **depth and clarity of the ocean** — deep navy backgrounds, electric cyan accents, fluid micro-animations, and frosted-glass UI elements. Every interaction is designed to feel as refreshing as cracking open a cold sparkling drink.

**Color Palette:**

| Name         | Hex       | Usage                  |
|--------------|-----------|------------------------|
| Deep Ocean   | `#03045e` | Backgrounds            |
| Ocean Blue   | `#0077b6` | Primary actions        |
| Sky Blue     | `#00b4d8` | Highlights & hover     |
| Aqua Fizz    | `#48cae4` | Accents & badges       |
| Foam White   | `#caf0f8` | Text & light elements  |
